### Proyecto 3 — Venta de páginas web automatizada ⭐ MUY IMPORTANTE
**Concepto completo:**
Sistema que busca negocios SIN página web y les propone una automáticamente.

**Flujo:**
1. Scraper busca comercios en Google Maps y directorios locales
2. Sistema envía email personalizado con propuesta + beneficios + precio
3. Si el cliente acepta → se crea su página con Landing Page v2
4. El cliente recibe acceso a un panel para modificar SU página
5. Una sola plataforma para todos los clientes — cada uno ve solo lo suyo

**Modelo de precios:**
- Pago único: creación + dominio 1 año → $150-$400 USD
- Renovación anual: solo el dominio → $15-$30 USD/año
- Opción premium: app de gestión con panel de edición → precio mayor
- Competencia cobra $500-$2000 → Mario es más accesible pero no regala nada

**Con 10 clientes/mes = $1500-$4000 USD/mes**
